Üniversite zamanında yapmış olduğum çalışma

1-) Kullanıcı Giriş Sistemi (Form1)<br>
-Kullanıcı adı ve şifre ile giriş<br>
-Şifreler MD5 ile hashleniyor<br>
-SQL Server üzerinden kimlik doğrulama<br>
-Başarılı girişte ana menüye yönlendirme<br>

2-) Kullanıcı Yönetimi (Form2)<br>
-Yeni kullanıcı ekleme (stored procedure: up_KullaniciEkle)<br>
-Kullanıcı silme<br>
-Kullanıcı bilgilerini güncelleme<br>
-Kullanıcı arama (kullanıcı adına göre)<br>
-Kullanıcı listesini görüntüleme (DataGridView)<br>

3-) Ana Sipariş Ekranı (GirisYapmis)<br>
-Pizza siparişi alma<br>
-Pizza boyutu seçimi (Küçük: 10 TL, Orta: 15 TL, Büyük: 20 TL)<br>
-Adet seçimi<br>
-Ekstra malzemeler (Sucuk, Sosis, Mantar, Kaşar, Peynir, Sebze)<br>
-İçecek seçenekleri (Kola, Fanta, Sprite)<br>
-Müşteri bilgileri (Ad, Soyad, Telefon, Adres)<br>
-Otomatik fiyat hesaplama<br>
-Sipariş listesi görüntüleme<br>
-Sipariş temizleme<br>
-Yazdırma desteği<br>
-Tarih/saat gösterimi<br>

4-) Müşteri Yönetimi (Musteriler)<br>
-Yeni müşteri ekleme (stored procedure: up_MusteriEkle)<br>
-Müşteri silme<br>
-Müşteri listesini görüntüleme<br>
-Müşteri bilgileri (Ad, Soyad, Telefon, Adres, Açıklama)<br>

5-) Toptancı Yönetimi (Toptanciler)<br>
-Yeni toptancı ekleme (stored procedure: up_ToptanciEkle)<br>
-Toptancı silme<br>
-Toptancı listesini görüntüleme<br>
-Toptancı bilgileri (Firma İsmi, Telefon, Adres, Kişi Adı, Kişi Telefonu)<br>

6-) Teknik Detaylar<br>
-Platform: Windows Forms (.NET Framework 4.5.2)<br>
-Veritabanı: SQL Server (GeceP3)<br>
-Bağlantı: Uzak SQL Server <br>
-Güvenlik: MD5 hash ile şifreleme<br>
-Veri Erişimi: Stored procedure ve doğrudan SQL sorguları<br>
-Dataset: 9 adet DataSet (GeceP3DataSet - GeceP3DataSet9)<br>

7-) Görsel Öğeler<br>
-Pizza görselleri (13 adet)<br>
-Pizza menü görselleri (5-6 adet)<br>
-Ek ürün görselleri (5 adet)<br>
-İçecek görselleri (3 adet)<br>
